# N199RF Pilot Reference

Static site for pilot training and avionics reference.

## Deploy to Vercel
1. Drag this folder to vercel.com/new
2. Or: `vercel` from this directory
3. Set custom domain to n199rf.chrisvanwinkle.info (or similar)

## Deploy to Netlify
1. Drag this folder to app.netlify.com/drop
2. Or: connect a GitHub repo

## Structure
- `/` — landing page
- `/course/` — main mastery course (with sync)
- `/g3x-pfd/` — G3X Touch PFD reference
- `/gtn750/` — GTN 750Xi reference
- `/gfc500/` — GFC 500 autopilot reference
- `/flight-1-flint-hills/` — flight playbook
